# GamesLibraryApp

A Kotlin Android app that lets users discover games (powered by the [RAWG API](https://rawg.io/apidocs), manage their personal game library, authenticate with Firebase, and share library items with other users.

### Features
   * Browse and search games from RAWG (game details, images, release dates, genres).
   * User sign-up / sign-in (Firebase Authentication â€” email/password and optional providers).
   * Personal game library: add/reomve entries from your library
   * Library sharing: look up other users libraries with their username
   * Real-time sync of library data using Firebase Realtime Database.
